## Branch Naming Convention

แนวทางการตั้งชื่อ branch ที่ใช้ในโปรเจกต์นี้:
```
<type>/<description>
```
### Types
- feature: new feature
- fix: bug fix
- chore: setup or maintenance
- docs: documentation

### Examples
- feature/frontend-map-ui
- feature/backend-dijkstra
- fix/backend-path-error
